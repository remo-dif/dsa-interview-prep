# DSA Interview Prep

This folder organizes the JavaScript solutions in the parent directory by the flow of *Learning JavaScript Data Structures and Algorithms, Third Edition* and by technical interview usefulness.

Use this in three passes:

1. Review the topic in the book.
2. Open the matching files in `CATALOG.md`.
3. Fix or rehearse the items marked in `REVIEW_NOTES.md`.

## Book-Aligned Sequence

1. JavaScript fundamentals and complexity
2. Arrays and strings
3. Stacks, queues, and deques
4. Linked lists
5. Sets, dictionaries, maps, and hashes
6. Recursion
7. Trees and binary search trees
8. Graphs
9. Sorting and searching
10. Algorithm design techniques: divide and conquer, dynamic programming, greedy, and backtracking

## Interview Priorities

Focus first on these high-frequency areas:

1. Arrays, strings, maps, and prefix sums
2. Linked lists with two pointers and dummy nodes
3. Recursion, backtracking, and divide and conquer
4. Trees and BFS/DFS traversal
5. Graph BFS/DFS
6. Dynamic programming
7. Sorting, binary search, and heaps

## Practice Routine

For each problem, prepare these five things:

1. Problem statement in your own words
2. Brute-force approach and why it is too slow
3. Optimized idea and key data structure
4. Time and space complexity
5. Three edge cases
